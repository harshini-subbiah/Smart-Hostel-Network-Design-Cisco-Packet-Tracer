# Smart Hostel Network Design using Cisco Packet Tracer

A Cisco Packet Tracer-based enterprise hostel network that demonstrates VLAN segmentation, DHCP, DNS, ACL security, wireless connectivity, server access control, internet simulation, and centralized network services. The project simulates a real-world hostel environment with secure communication between different user groups while enforcing network security policies.

---

## Features

- VLAN Segmentation for different hostel floors and departments
- Router-on-a-Stick Inter-VLAN Routing
- DHCP Server for automatic IP address assignment
- DNS Server for hostname resolution
- HTTP Web Server hosting internal webpages
- Guest Wi-Fi Network
- Access Control Lists (ACLs) for network security
- Wireless Connectivity
- Server Access Restrictions
- Switch Trunk Configuration
- Internet Connectivity Simulation
- Syslog Server for logging and monitoring

## How to Run

1. Install Cisco Packet Tracer.
2. Clone this repository.

```bash
git clone https://github.com/yourusername/Smart-Hostel-Network-Design-Cisco-Packet-Tracer.git
```

3. Open the following file:

```
PacketTracer/hostel.pkt
```

4. Wait for the simulation to initialize.

5. Switch between **Realtime** and **Simulation** mode as needed.
